# Introduction

![PySilon Next Banner](../static/img/Banner.jpeg)

**PySilon-Next** is an advanced Remote Administration Tool (RAT) written in Python, fully controllable through Discord with a dedicated GUI builder to make preparation easier.

It is a modern continuation of the legacy PySilon project, updated for the latest Python versions and enhanced with new features.

## About

PySilon-Next takes the original codebase to the next level by:
*   Updating dependencies to their latest secure versions.
*   Adding support for Python 3.10+.
*   Implementing new features while maintaining core functionality.
*   Providing a dedicated GUI builder.
