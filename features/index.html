Features

PySilon-Next allows you to control a target PC entirely through Discord.

System Control​

  • Multi-PC Handling: Control multiple victims simultaneously.
  • UAC Bypass: Gain administrative permissions on startup.
  • Self-Destruct: Delete the RAT remotely to clean tracks.
  • Shell Access: Run CMD and Powershell commands.
  • File Management: Browse, upload, download, and remove files.
  • Process Management: View, kill, disable, blacklist, and whitelist processes.
  • Registry Editing: Manipulate the Windows Registry.
  • Power Options: Shutdown, Reboot, or Logoff the target.

Surveillance​

  • Keylogger: Log every key pressed.
  • Screenshots: Take screenshots on demand.
  • Screen Recording: Record the screen for a set duration.
  • Webcam: Take photos using the connected webcam.
  • Microphone: Record audio 24/7 or stream live to a voice channel.
  • Data Grabbers:
    • WiFi Passwords
    • Browser History, Cookies, and Passwords
    • Discord Tokens
    • System Information

Trolling & Fun​

  • Blue Screen of Death (BSOD): Trigger a system crash.
  • Fork Bomb: Overload the system with processes.
  • Graphics Manipulation: Glitch the screen, draw bitmaps, or display text.
  • Monitor Control: Turn monitors off and on.
  • Input Blocking: Block mouse and keyboard input.
  • Text-To-Speech: Speak messages on the target PC.
  • Audio Playback: Play audio files in the background.
  • Jumpscares: Play loud videos or images.
  • Website Blocking: Prevent access to specific websites.

Advanced Features​

  • Critical Process: Elevates the process so that killing it triggers a BSOD.
  • Crypto Clipper: Replaces copied crypto wallet addresses with yours.
  • Anti-VM: Detects VirtualBox, VMWare, etc., and refuses to run to avoid analysis.
  • Debug Mode: Specialized mode for easier testing and contribution.

Premium Features (Coming Soon)​

  • Maintained modules for supporters.
  • Advanced evasion techniques.
  • More powerful grabbers.
